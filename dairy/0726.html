선택자

  1. 하위선택자(한칸 띄우기) 안 : 요소 안의 항목 다
  2. 자식선택자(>) 안 : < > 안에 직계 아래것
  3. 인접선택자(+) 다음 : 다음에 오는(포함관계x) 가장 가까운거 하나
  4. 형제선택자(~) 다음 : 다음(뒤)에 오는 것 모두, 건너뛰어도(포함관계x)
  5. >* : 하위 항목 모두(본인 포함x)
  6. 그룹선택자 , : 여러개 선택
  • 띄우기, > : 안(포함관계 o)
  • +, ~ : 다음(포함관계 x)
