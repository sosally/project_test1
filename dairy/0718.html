태그 정리

  • blockquote : 인용문
  • 이메일 : a href="mailto"
  • 전화번호 : a href="tel:010-"
  • 문자 : a href="sms:031"
  • 기울임 : em, i, address
  • 볼드 : strong, b
  • 형광펜 : mark
  • id : 특별관리를 위한 이름 붙이기
  • a 태그 안에
    • title : 하버링시 설명, 모든 태그에 사용가능 ex.새창열기
    • target : _blank (새창열림 달아주기)
